# OpenAPI specifications for JSON Web Signatures (JWS) and JSON Web Encryption (JWE) 

This repository contains OpenAPI specifications for JSON Web Signatures (JWS) and JSON Web Encryption (JWE)
using the JSON Serializations. 

## Background
While JWS and JWE are most commonly used with Compact Serialization, both standards also support JSON Serialization 
which can be integrated into applications that rely on structured JSON formats
for message exchange and data storage.
